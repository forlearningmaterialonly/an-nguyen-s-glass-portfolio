# An Nguyen — Portfolio

A glassmorphism portfolio site built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Features
- Glassmorphism UI with aurora gradient background
- Animated particle background (canvas, respects reduced-motion)
- Typewriter effect in the hero
- Scroll-reveal animations + staggered page-load reveal
- Fully responsive, mobile-first
- Accessible: semantic landmarks, ARIA labels, focus states

## Sections
Hero · About · Projects · Skills · Experience · Contact

## Setup
```bash
npm install
npm run dev
```

Open http://localhost:5173.

## Customize
- Personal copy: edit components in `src/components/portfolio/`
- Design tokens (colors, gradients, glass surfaces): `src/index.css`
- Tailwind theme & animations: `tailwind.config.ts`
- Page composition: `src/pages/Index.tsx`

## Build
```bash
npm run build
npm run preview
```
